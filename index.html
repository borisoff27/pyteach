Создание исполняемого или бинарного файла

Для запуска готового приложения, написанного на языке Python, его необходимо скомпилировать / переконвертировать в отдельный файл. Сделать это можно по-разному, рассмотрим один из способов: pyinstaller
Установка pyinstaller:
pip3 install pyinstaller
После установки в терминале / консоли необходимо написать следующую команду:
pyinstaller test.py
Исполняемый файл будет создан в папке под названием dist в директории вашей программы Python. Однако при таком методе будет генерироватсья множество зависимостей.
Чтобы создать один исполняемый файл, команда далжна быть стать следующего вида:
pyinstaller --onefile test.py
Но и это ещё не всё! Всякий раз при запуске будет появляться окно консоли, которое можно спрятать, используя флажки -w или –noconsole:
pyinstaller -w test.py
Pyinstaller предоставляет множество вариантов для упаковки вашего приложения, чтобы увидеть полный список, используйте --help:
pyinstaller --help
